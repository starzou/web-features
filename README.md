# web-features
### 探究各种Web前沿技术，在企业级开发中的最佳实践。
### Easy to use.


## Get
```bash  
$ git clone https://github.com/starzou/web-features  
$ cd web-features  
```

## Install Packages
```bash  
$ npm install && bower install
```

## Usage
```bash  
$ npm start
```
## Visit the website
```
http://127.0.0.1:9000
```
